Tabuada em Python
Descrição:Script que calcula e exibe a tabuada de um número de 1 a 10.
Desenvolvido como exercício prático no curso de Lógica de Programação - Senac.
Como Usar:
1. Certifique-se que tem Python instalado
2. Execute: `python exercicio2.py`
3. Digite um número quando solicitado
4. Veja a tabuada ser exibida (de 1 a 10)

Tecnologias
- Python 3.x
- Conceitos: loops (for), input, conversão de tipos (int)
